# CareerPilot Chrome Extension

Save jobs from LinkedIn, Naukri, Indeed, Glassdoor, Instahyre, and Cutshort to your CareerPilot Tracker.

## Install (Developer Mode)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this `chrome-extension` folder

## Usage

1. Browse a job listing on a supported portal
2. Click the CareerPilot extension icon
3. Click **Save to Tracker**

Saved jobs are stored in extension storage. Sync with the main app via Settings → Import (coming soon) or manually add to Tracker.

## Icons

Add `icon48.png` and `icon128.png` to this folder for Chrome Web Store publishing.
